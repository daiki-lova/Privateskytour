# SkyView デザインガイドライン & 使用書

東京ヘリポート発のラグジュアリー・ヘリコプター遊覧予約システム「SkyView」のデザインおよび実装ガイドラインです。

## 1. ブランドアイデンティティ
洗練、信頼、そして特別感を表現するため、グラデーションを廃したフラットでミニマルなデザインを採用しています。ライトモードに最適化されており、高級ホテルのような静謐な印象を与えます。

### カラーパレット
| 用途 | カラー | コード | CSS変数 |
| :--- | :--- | :--- | :--- |
| プライマリー（基本色） | ネイビー | `#003366` | `--primary` |
| アクセント（強調・UI） | ビビッドブルー | `#0066FF` | `--vivid-blue` |
| 背景 | 純白 | `#FFFFFF` | `--background` |
| テキスト（主要） | 漆黒 | `oklch(0.145 0 0)` | `--foreground` |
| 境界線・補助 | ニュートラルグレー | `rgba(0, 0, 0, 0.1)` | `--border` |

## 2. タイポグラフィ
可読性と高級感を両立させるため、標準的なフォントサイズとウェイトを厳格に管理しています。

- **Heading 1 (h1)**: `text-2xl` / `font-medium` (24px相当)
- **Heading 2 (h2)**: `text-xl` / `font-medium` (20px相当)
- **Body (p)**: `text-base` / `font-normal` (14px/16px相当)
- **Button / Label**: `text-base` / `font-medium`

## 3. コンポーネント設計方針

### 共通ルール
- **角丸 (`--radius`)**: `0.625rem` (10px) を基本とし、モダンで柔らかな印象を与えます。
- **グラデーション**: 使用禁止。単色（ソリッドカラー）のみを使用し、洗練された高級感を維持します。
- **レスポンシブ**: モバイル・デスクトップ両対応（モバイルファースト推奨）。

### 特徴的な機能ロジック
- **アルファード送迎フォーム**: 30分以上のフライトプランが選択された際、動的に「高級送迎（アルファード）」のオプションフォームを表示します。
- **ページ集約型アーキテクチャ**: Next.jsへの移行を容易にするため、主要なページロジックは `/components/pages/` に集約しています。

## 4. ページ構成とルーティング
現在は `/App.tsx` 内でステート管理による疑似ルーティングを行っています。

- **LandingPage (`/components/pages/LandingPage.tsx`)**: 刷新されたトップページ。
- **TourDetailPage (`/components/pages/TourDetailPage.tsx`)**: 各プランの詳細。
- **BookingPage (`/components/pages/BookingPage.tsx`)**: 3ステップの予約ウィザード。

## 5. 実装上の注意点
- **画像**: 画像は必ず `ImageWithFallback` コンポーネントを使用し、Unsplashから最適なラグジュアリー画像を動的に取得してください。
- **アイコン**: `lucide-react` を使用し、一貫性を持たせます。
- **アニメーション**: `motion/react` を使用し、控えめで優雅なフェードイン・フェードアウトを中心に構成します。
